Portfolio

Tresor-Sicherheitssystem mit Zweifaktor-Authentifizierung

Bei dieser Projektarbeit handelt es sich um eine Arduino-Anwendung, welche ein Sicherheitssystem mit Zweifaktor-Authentifizierung für ein Türschloss darstellen soll. Diese Applikation habe ich in Zusammenarbeit mit 2 Klassenkameraden erstellt. Zur Realisierung haben wir einen Simulator namens Tinkercad benutzt.
Hier der Link zum Schaltkreis, um es mal selber auszuporbieren.

SQL-Hangman

Wie der Name eigentlich schon verrät handelt es sich hierbei um das beliebte Spiel Hangman, welches ich mittels C# erstellt habe. Mit dem kleinen Unterschied, dass die Begriffe anstatt aus einer einfachen Textdatei, aus einer MySql Datenbank gelesen werden.

Autorennspiel in WinForms

Dies ist eines der ersten Programmierprojekte, die ich an der BBBaden erstellt habe. Es handelt sich um ein "Vertikal scrollendes Spiel", in dem der Spieler ein Auto an anderen Fahrzeugen vorbeisteuern muss.

Multiple Choice Quiz

Hierbei handelt es sich um ein Multiple Choice Quiz, welches das Allgemeinwissen des Nutzers testen soll. Die Fragen werden aus einem Textdokument herausgelesen und dann in zufälliger Reihenfolge abgefragt. Hierbei handelt es sich ebenfalls um ein Windows Forms Projekt.

Countdown Timer

Diese JavaScript Applikation ermöglicht es dem Benutzer zu sehen, wie viel Zeit noch zu einem bestimmten Zeitpunkt zu vergehen ist. Dafür kann er das gewünschte Datum und die gewünschte Uhrzeit eingeben. Dieses Projekt war ebenfalls eine Gruppenarbeit
